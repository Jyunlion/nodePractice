# nodePractice
無法安裝 node23.1.0時 需先安裝 nvs 
# nvs 安裝
git clone https://github.com/jasongin/nvs.git ~/.nvs 後
找到 自己本機端 nvs.sh 的檔案位置，將 ./RefDocument/nodeVersionControl.sh 內的檔案位置修改後執行
# 透過 nvs 安裝 node 23.1.0
nvs add 23.1.0
node -v 確認版本，及安裝成功即可
npm start
